# deep-interview

Use `kch deep-interview` when the task is broad, underspecified, or likely to branch.

```bash
kch deep-interview "Improve the runtime" --json
```

If the result is blocked, answer the listed questions before launching `kch plan`, `kch ralph`, or `kch team`.
